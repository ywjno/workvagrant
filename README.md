# WorkVagrant

It contains `git`, `postgresql` and `oracle java 6` recipes.

## How to use

```bash
git clone git://github.com/ywjno/workvagrant.git
cd workvagrant
bundle install
rake install
rake up
```

run `rake -T` find other command.
